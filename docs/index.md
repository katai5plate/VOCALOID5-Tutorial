# VOCALOID5-Tutorial
VOCALOID5に慣れるためのメモ

## ATTACK EFFECT

### PITCH
![attack_pitch](image/attack_pitch.png)<br>
<audio src="sound/attack_pitch.mp3" controls></audio><br>

#### Down
-  1：半音上から下がる
-  2：全音上から下がる
-  3：１オクターブ上から下がる

#### Enka
- 演歌風に揺れる

#### Up
-  1：半音下から上がる
-  2：全音下から上がる
-  3：３音下から上がる
-  4：７音下から上がる
-  5：１オクターブ下から上がる

### DYNAMICS
![attack_dyn](image/attack_dyn.png)<br>
<audio src="sound/attack_dyn.mp3" controls></audio><br>

#### Accent
-  1：PITCH-UPにアクセントが＋
-  2：力む
-  3：より強く力む

#### Crescendo
-  1：だんだん強く「うー」
-  2：だんだん強く「あー」

#### DoubleAccent
-  1：よりアクセントがつく
-  2：1より自然に
-  3：1より強く

### TONE
![attack_tone](image/attack_tone.png)<br>
<audio src="sound/attack_tone.mp3" controls></audio><br>

#### Airy
-  1：ウィスパーボイス
-  2：かすれ声

#### Clear
- 透き通る声

#### Growly
-  1：全音上から下がるグロウル
-  2：つらそうなグロウル
-  3：大声なグロウル「があー」
-  4：４音下から上がるグロウル「うあー」

#### (Pronounce)
-  Narrowed：口をすぼめる「んー」
-  Oh-uh：曖昧に口を狭める「んうー」
-  Opend：口を開く
-  Wow：口を徐々に狭める「ぉわーぉ」

### ORNAMENTATION
![attack_ornam](image/attack_ornam.png)<br>
<audio src="sound/attack_ornam.mp3" controls></audio><br>

#### (Throat)
-  Catty：９音上から鳴き声のように下がる「んゃぁーぁ」
-  Floggy：全音下からイの口で鳴くように上げる「うぃぃー」
-  Foxy：全音下から吠えるように「わぁっぁー」
-  Piggy：７音上から悶えるように下がる「むぅぅぁー」

#### Trill
- ９音下から高速ビブラートで上がる「ぁﾊﾊﾊﾊﾊぁー」

#### VocalFry
-  F Power：喉を鳴らす「ｹﾞあー」
-  F soft：開き目に喉を鳴らす「ｶﾞあー」
-  M：閉じ目に喉を鳴らす「ｳﾞあー」

## RELEASE EFFECT

### PITCH
![rel_pitch_1](image/rel_pitch_1.png)<br>
<audio src="sound/rel_pitch_1.mp3" controls></audio><br>
![rel_pitch_2](image/rel_pitch_2.png)<br>

#### Down
-  1：半音下がる
-  2：全音下がる
-  3：１オクターブ下がる

#### Melisma
-  1：メリスマ唱法　-5 -> -2
-  2：メリスマ唱法　+1 -> -3 -> -1
-  3：メリスマ唱法　-6 -> -4 -> -6 -> -9
-  4：メリスマ唱法　-2 -> -7 -> -6 -> -7
-  5：メリスマ唱法　-2 -> -3 -> -5 -> -3 -> -5

#### Up
-  1：半音上がる
-  2：全音上がる
-  3：１オクターブ上がる

#### Vib
-  Down：ビブラートしつつ下がる
-  Down 2：強くビブラートしつつ下がる
-  Extreme：「あ～れ～」ぐらい強くビブラート
-  Extreme_F.O：上の振幅を徐々に減らしていく
-  Fast：わずかに早め
-  Fast_F.O：上の振幅を徐々に減らしていく
-  Normal：普通ぐらい
-  Normal_F.O：上の振幅を徐々に減らしていく
-  Slight：ひかえめ
-  Slight_F.O：上の振幅を徐々に減らしていく
-  Soft：やわらかに息を吹き込んで「ふ～」
-  Up：ビブラートしつつ上がる
-  UpDown：ビブラートしつつ上がって急降下

#### Uh-wa

### ORNAMENTATION
![rel_ornam](image/rel_ornam.png)<br>
<audio src="sound/rel_ornam.mp3" controls></audio><br>

#### (Finish)
-  Breathe_Out：途中で喉笛を止め吐息を出す「あーﾊｧ」
-  Stop：途中で声をきっぱり止める
-  Stop_HE：途中でヘッという声で止める「あーｯﾍｯ」
-  Stop_W：途中で口をすぼめて止める「あーぅ」

#### (Speed-Vib)
-  Tremolo：笑うように高速ビブラートしつつ１０音下がる
-  Trill：笑うように高速ビブラートしつつ早めに１オクターブ下がる

#### VocalFry
-  F Power：最後に強く喉を鳴らす「あーｩｹﾞ」
-  F soft：最後に喉を鳴らす「あーｴﾞ」
-  M：最後に弱く喉を鳴らす「あーｳﾞ」
