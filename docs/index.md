## ATTACK EFFECT
### PITCH

#### Down
1. _：半音上から下がる
2. 2：全音上から下がる
3. 3：１オクターブ上から下がる
#### Enka
：演歌風に揺れる
#### Up
1. _：半音下から上がる
2. 2：全音下から上がる
3. 3：３音下から上がる
4. 4：７音下から上がる
5. 5：１オクターブ下から上がる

#### Crescendo
1. _：だんだん強く「うー」
2. 2：だんだん強く「あー」

### TONE
![attack_tone](image/attack_tone.png)<br>
<audio src="sound/attack_tone.mp3" controls></audio><br>
#### Airy
1. _：ウィスパーボイス
2. 2：かすれ声
#### Clear
：透き通る声
#### Growly
1. _：全音上から下がるグロウル
2. 2：つらそうなグロウル
3. 3：大声なグロウル「があー」
4. 4：４音下から上がるグロウル「うあー」
#### (Pronounce)
1. Narrowed：口をすぼめる「んー」
2. Oh-uh：曖昧に口を狭める「んうー」
3. Opend：口を開く
4. Wow：口を徐々に狭める「ぉわーぉ」
### ORNAMENTATION
![attack_ornam](image/attack_ornam.png)<br>
<audio src="sound/attack_ornam.mp3" controls></audio><br>
#### (Throat)
1. Catty：９音上から鳴き声のように下がる「んゃぁーぁ」
2. Floggy：全音下からイの口で鳴くように上げる「うぃぃー」
3. Foxy：全音下から吠えるように「わぁっぁー」
4. Piggy：７音上から悶えるように下がる「むぅぅぁー」
#### Trill
：９音下から高速ビブラートで上がる「ぁﾊﾊﾊﾊﾊぁー」
#### VocalFry
1. F Power：喉を鳴らす「ｹﾞあー」
2. F soft：開き目に喉を鳴らす「ｶﾞあー」
3. M：閉じ目に喉を鳴らす「ｳﾞあー」

#### Down
1. _：半音下がる
2. 2：全音下がる
3. 3：１オクターブ下がる
#### Melisma
1. _：メリスマ唱法　-5 -> -2
2. 2：メリスマ唱法　+1 -> -3 -> -1
3. 3：メリスマ唱法　-6 -> -4 -> -6 -> -9
4. 4：メリスマ唱法　-2 -> -7 -> -6 -> -7
5. 5：メリスマ唱法　-2 -> -3 -> -5 -> -3 -> -5
#### Up
1. _：半音上がる
2. 2：全音上がる
3. 3：１オクターブ上がる
#### Vib
1. Down：ビブラートしつつ下がる
2. Down 2：強くビブラートしつつ下がる
3. Extreme：「あ～れ～」ぐらい強くビブラート
4. Extreme_F.O：上の振幅を徐々に減らしていく
5. Fast：わずかに早め
6. Fast_F.O：上の振幅を徐々に減らしていく
7. Normal：普通ぐらい
8. Normal_F.O：上の振幅を徐々に減らしていく
9. Slight：ひかえめ
10. Slight_F.O：上の振幅を徐々に減らしていく
11. Soft：やわらかに息を吹き込んで「ふ～」
12. Up：ビブラートしつつ上がる
13. UpDown：ビブラートしつつ上がって急降下

### ORNAMENTATION

#### (Finish)
1. Breathe_Out：途中で喉笛を止め吐息を出す「あーﾊｧ」
2. Stop：途中で声をきっぱり止める
3. Stop_HE：途中でヘッという声で止める「あーｯﾍｯ」
4. Stop_W：途中で口をすぼめて止める「あーぅ」
#### (Speed-Vib)
1. Tremolo：笑うように高速ビブラートしつつ１０音下がる
2. Trill：笑うように高速ビブラートしつつ早めに１オクターブ下がる
#### VocalFry
1. F Power：最後に強く喉を鳴らす「あーｩｹﾞ」
2. F soft：最後に喉を鳴らす「あーｴﾞ」
3. M：最後に弱く喉を鳴らす「あーｳﾞ」
